# Influx Agent
Influx Agent is a monitoring tool used to send data to Influx Core.

### Prerequisites
- NodeJS v16.x+
- NPM v9.x+
- Yarn v1.x+
- Git
- Linux-based system

### Installation
```
git clone https://github.com/cmrxnn/influx-agent
cd influx-agent
yarn
yarn start
```
